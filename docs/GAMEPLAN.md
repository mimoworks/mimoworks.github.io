# Game plan

## Objective
Launch a clean v1 website that makes **mimoworks** credible, showcases **Lumofield** first, and leaves room for **mimo connect** later.

## Launch philosophy
- ship a strong static marketing site first
- do not build ecommerce, backend, or advanced tooling yet
- optimize for credibility, clarity, and future extensibility

## Phase 1: foundation
### Deliverables
- Astro project scaffold
- shared layout
- design tokens
- reusable sections and cards
- responsive navigation and footer

### Success criteria
- brand shell feels coherent
- pages are easy to extend
- mobile-first layout works well

## Phase 2: content and pages
### Build these pages
1. Home
2. Lumofield
3. Connect
4. About
5. Contact

### Success criteria
- Home clearly prioritizes Lumofield
- Connect is visible but secondary
- About explains mimoworks as umbrella brand
- Contact provides inquiry path

## Phase 3: polish
### Add
- metadata and Open Graph tags
- favicon and social preview placeholders
- 404 page
- basic motion only if subtle
- accessibility pass

### Do not add yet
- cart
- checkout
- CMS
- login
- advanced forms provider

## Phase 4: deployment
### Start
- GitHub Pages
- GitHub Actions build + deploy

### Later
- move to custom domain
- update site URL and canonical metadata

## Recommended information hierarchy on homepage
1. mimoworks hero / positioning
2. Lumofield feature section
3. Lumofield product preview
4. mimo connect teaser
5. about / process / founder values
6. contact / interest CTA

## What the site should make a visitor feel
### On home page
- this brand feels thoughtful
- Lumofield looks real and intentional
- there is more to come beyond lamps

### On Lumofield page
- warm, calm, premium but not flashy
- product line feels lifestyle and home-oriented

### On Connect page
- practical, clear, early-stage but credible
- same company, different expression

## Kill list
Avoid these traps:
- overbuilding before products are ready
- making each line look like a separate company
- cluttered nav and too many categories
- placeholder lorem ipsum surviving too long
- technical jargon dominating the copy
